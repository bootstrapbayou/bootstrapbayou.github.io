Welcome to CCI School Labs

Delve into hands-on labs focused on Azure services and infrastructure. Labs are designed to build from foundational skills to advanced solutions, providing continuous learning and skill development in cloud technologies.

Lab Categories

Choose a category to explore labs related to Azure cloud infrastructure, automation, security, and more. Each category offers labs that build practical skills needed for managing enterprise-grade cloud solutions.

Infrastructure Setup

Set up and configure virtual networks, virtual machines, and other core infrastructure elements on Azure.

Cloud Automation

Automate Azure resource management, deployments, and other processes using Azure Automation and PowerShell.

Security & Compliance

Explore Azure security tools, compliance policies, and identity management to secure cloud environments.

Infrastructure Setup Labs

Azure Virtual Network Configuration

Set up and configure Azure Virtual Networks, subnets, and peering to enable secure communication between cloud resources.

Deploying Virtual Machines

Launch and manage virtual machines on Azure, including scaling, networking, and performance tuning.

Azure Storage Solutions

Explore Azure Storage accounts, configure blob storage, and manage access policies for secure data storage.

Cloud Automation Labs

Automating Deployments with ARM Templates

Use Azure Resource Manager (ARM) templates to automate deployments of Azure resources, ensuring consistency and efficiency.

Azure Logic Apps Automation

Automate workflows across applications using Azure Logic Apps, integrating services for seamless cloud operations.

Managing Resources with Azure CLI

Manage Azure resources using Azure CLI commands, streamlining administration and configuration tasks.

Security & Compliance Labs

Azure Active Directory (AAD) Configuration

Set up and manage Azure Active Directory, configure single sign-on (SSO), and ensure secure identity management.

Configuring Network Security Groups (NSGs)

Implement Network Security Groups to control inbound and outbound traffic and enhance network security.

Azure Key Vault Management

Manage sensitive information using Azure Key Vault, including secrets, keys, and certificates.
