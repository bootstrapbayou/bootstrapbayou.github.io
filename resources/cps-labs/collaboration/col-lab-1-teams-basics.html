Microsoft Teams Basics

This hands-on lab is designed to provide foundational skills in Microsoft Teams setup, channel creation, and basic feature exploration to enhance team collaboration within a modern enterprise environment.

**Please review all the information in the Lab Overview Table below before visiting the GitHub repository.** Understanding each objective, prerequisite, and expected outcome is essential for a successful lab experience. Thoroughly reviewing these details will help ensure you are prepared to engage with the lab material and make the most of the resources provided in the GitHub repository.

Duration

1.5 hours

Prerequisites

Microsoft 365 credentials, basic understanding of collaboration tools

GitHub Repository

Click here to access the full lab details and additional resources on GitHub.
Lab Overview
Title Microsoft Teams Basics
Description Set up Microsoft Teams, create channels, and explore basic features to enhance team collaboration. This lab will provide hands-on experience with foundational elements of Microsoft Teams.
Objective To familiarize users with the basic functionalities of Microsoft Teams, enabling effective team communication and collaboration.
Scope Includes setting up a new Microsoft Teams environment, creating channels, managing permissions, and exploring chat, video, and file-sharing features.
Stakeholders IT administrators, team leads, faculty members, and university staff who require collaboration tools for academic and administrative tasks.
Lab Activities
Preparation
  • Ensure access to Microsoft Teams via Microsoft 365 credentials.
  • Set up a basic Teams environment with the necessary permissions.
  • Review basic Teams documentation for channel creation and user management.
Tasks
  • Create a new Microsoft Team for a department or project.
  • Set up channels within the team, including standard and private channels.
  • Explore chat, file sharing, and meeting scheduling features.
  • Manage user permissions and access controls within Teams.
Expected Outcomes
  • Successfully set up a new Microsoft Team with appropriate channels.
  • Understand basic Teams features like chat, file sharing, and meeting scheduling.
  • Efficient management of users and permissions within Teams.
Implementation Plan
Methodology Hands-on approach focusing on individual tasks related to Teams setup, channel management, and communication features.
Tools & Technologies Microsoft Teams, Microsoft 365 credentials, and basic knowledge of collaboration tools.
Governance & Compliance
Governance Structure IT administrators oversee the setup process, ensuring compliance with university collaboration policies.
Compliance Adheres to university collaboration guidelines, ensuring data privacy and secure communication.
Documentation
Technical Documentation Step-by-step guide for Teams setup, channel creation, user management, and communication tools.
User Documentation User manuals, FAQs, and quick-start guides to help users understand the basics of Microsoft Teams.
Lab Closure
Final Review Review successful completion of Teams setup, gather user feedback, and identify areas for improvement.
Feedback Mechanism
  • Create a GitHub Gist summarizing the lab results.
  • Complete the Microsoft Form for lab feedback, including the link to the Gist.
